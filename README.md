<h1 align="center" style="border-bottom: none">
    ⭐️ Travel Agency Database Project GR ⭐️ 
</h1>

This project was made as a class assignment for the course "Databases II" at the University of West Attica former known as TEI of Athens. It's a travel agency website in greece that you "book" your ticket for a destination that you want. It uses mysql database to store users and destinations. Also this project is translated into greek.
